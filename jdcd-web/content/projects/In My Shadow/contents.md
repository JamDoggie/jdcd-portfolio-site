Starting development in late 2025, I wanted to capture a lot of what I liked about both horror games and movies. **_Tank Horror_** is a genre that has always interested me as a game designer. What started as an idea for a simple PSX style tank horror game has evolved into much more. This project so far has kick-started a brand new C# implementation of FMOD + Steam Audio support for Godot, and has evolved my art skills immensely.

In My Shadow is a psychological tank horror game set **_somewhere deep in rural Florida_**. The goal is to utilize the full potential of tank horror, and draw as much inspiration as I can from film. One other important thing for me with this project is to try and utilize directional sound and sound design as effectively as possible. This project has also been used as an excuse to improve upon the Spooky Manor Level Editor (see below).
