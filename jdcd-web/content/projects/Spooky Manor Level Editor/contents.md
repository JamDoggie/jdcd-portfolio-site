This is an [open-source](https://git.mappack.net/spkymnr/spkymnr-level-editor) level editor pipeline for <u>**working with both Blender and Godot**</u>. Working primarily within a Blender addon, the Spooky Manor Level Editor allows you to do all of the following:

- Import game assets into Blender.
- Drag & drop Godot materials on to level geometry in Blender.
- Author collision meshes for level geometry, and manipulate object properties.
- Import your level into the Godot Editor as a sort of "in-between" process to add nodes in post.
- Edit maps for a game you do not have the source code for.
- Export and pack .pck content files that can be loaded from the exported map binary.
- Export a standalone fully-working map file that can be loaded independently from the game code.

It is currently still being heavily worked on, refactored, and there is a lot of stuff that needs to be done for a lot of workflows. With that being said, I use this for all my Godot games and develop it alongside whatever I am working on!
